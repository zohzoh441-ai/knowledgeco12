📊 MS Excel — All Formulas
Har formula ka naam, kaam, aur example — beginners se advanced tak

➕ Math & Basic Formulas

Math

SUM — Jama karna

Cells ki values ko jama karta hai.

=SUM(A1:A10)
Example: A1=100, A2=200, A3=300 → =SUM(A1:A3) = 600
Math

AVERAGE — Ausat Nikalna

Numbers ka average nikalti hai.

=AVERAGE(A1:A10)
Example: 10,20,30 → =AVERAGE(A1:A3) = 20
Math

MAX — Sab se Bara

Range mein sab se badi value deta hai.

=MAX(A1:A10)
Example: 5,15,9,22,3 → =MAX(A1:A5) = 22
Math

MIN — Sab se Chota

Range mein sab se choti value deta hai.

=MIN(A1:A10)
Example: 5,15,9,22,3 → =MIN(A1:A5) = 3
Math

COUNT — Girna

Sirf numbers wali cells ko ginta hai.

=COUNT(A1:A10)
Example: A1=5, A2="Ali", A3=9 → =COUNT(A1:A3) = 2
Math

COUNTA — Sab Count

Khali nahi wali sari cells girnti hai.

=COUNTA(A1:A10)
Example: A1=5, A2="Ali", A3="" → =COUNTA(A1:A3) = 2
Math

PRODUCT — Zarb

Sari values ko aapas mein multiply karta hai.

=PRODUCT(A1:A3)
Example: A1=2, A2=3, A3=4 → =PRODUCT(A1:A3) = 24
Math

MOD — Baqiya

Taqseem ka baqiya (remainder) deta hai.

=MOD(number, divisor)
Example: =MOD(10, 3) = 1
Math

POWER — Quwwat

Kisi number ko power mein uthata hai.

=POWER(number, power)
Example: =POWER(2, 8) = 256
Math

SQRT — Jazar

Square root nikalti hai.

=SQRT(number)
Example: =SQRT(144) = 12
Math

ABS — Mutlaq Qeemat

Negative number ko positive bana deta hai.

=ABS(number)
Example: =ABS(-75) = 75
Math

ROUND — Takhmina

Number ko decimal places tak round karta hai.

=ROUND(number, num_digits)
Example: =ROUND(3.14159, 2) = 3.14
Math

ROUNDUP / ROUNDDOWN

Upar ya neeche round karta hai.

=ROUNDUP(3.11, 1) → 3.2
=ROUNDDOWN(3.99, 1) → 3.9
Use: Price ya percentage display ke liye.
Math

INT — Poora Adad

Decimal part hata ke sirf integer deta hai.

=INT(number)
Example: =INT(7.9) = 7
Math

SUMIF — Shart ke Sath Jama

Sirf wahi cells jama karta hai jo shart puri karein.

=SUMIF(range, criteria, sum_range)
Example: =SUMIF(A1:A10,">50",B1:B10) — Jahan A>50 wahan B ki sum
Math

SUMIFS — Multiple Conditions

Kai shart ke sath sum karta hai.

=SUMIFS(sum_range, range1, crit1, range2, crit2)
Example: Sales >1000 AND City="Lahore" wali sum
Stats

MEDIAN — Darmiyan

Middle value nikalti hai.

=MEDIAN(A1:A10)
Example: 1,2,3,4,5 → Median = 3
Stats

MODE — Sab se Zyada

Sab se zyada baar aane wali value.

=MODE(A1:A10)
Example: 1,2,2,3,4 → Mode = 2
Stats

STDEV — Inharaf

Standard deviation nikalti hai.

=STDEV(A1:A10)
Use: Data ki variation ya spread dekhne ke liye.
Stats

COUNTIF — Shart ke Sath Count

Sirf wahi cells jo criteria match karein.

=COUNTIF(range, criteria)
Example: =COUNTIF(A1:A10,"Pass") = Kitne Pass hain

🔤 Text Formulas

Text

CONCATENATE / CONCAT

Do ya zyada texts ko mila deta hai.

=CONCAT(A1," ",B1)
=A1&" "&B1
Example: "Tahir" + "Saeed" → Tahir Saeed
Text

LEFT — Bayi Taraf

Text ke left se characters leta hai.

=LEFT(text, num_chars)
Example: =LEFT("Faisalabad", 6) = Faisal
Text

RIGHT — Dayi Taraf

Text ke right se characters leta hai.

=RIGHT(text, num_chars)
Example: =RIGHT("Faisalabad", 4) = abad
Text

MID — Darmiyan Se

Text ke beech se characters nikalti hai.

=MID(text, start_num, num_chars)
Example: =MID("Faisalabad",7,3) = aba
Text

LEN — Lambai

Text ki length (characters count) deta hai.

=LEN(text)
Example: =LEN("Pakistan") = 8
Text

UPPER / LOWER / PROPER

Text ka case change karta hai.

=UPPER("tahir") → TAHIR
=LOWER("TAHIR") → tahir
=PROPER("tahir saeed") → Tahir Saeed
Use: Names ya headings ko format karne ke liye.
Text

TRIM — Fazool Space Hatao

Aage peeche ki extra spaces hata deta hai.

=TRIM(text)
Example: =TRIM(" Tahir ") = "Tahir"
Text

FIND / SEARCH

Text mein kisi character ki position dhundh ta hai.

=FIND("a", "Pakistan")
Example: =FIND("k", "Pakistan") = 4 (position)
Text

REPLACE / SUBSTITUTE

Text ka koi hissa badal deta hai.

=SUBSTITUTE("Hello World","World","Pakistan")
Result: Hello Pakistan
Text

TEXT — Format Karo

Number ko formatted text mein badlta hai.

=TEXT(A1,"#,##0.00")
=TEXT(TODAY(),"DD/MM/YYYY")
Example: =TEXT(1500,"$#,##0") = $1,500

⚡ Logical Formulas

Logical

IF — Agar

Shart check karta hai aur result deta hai.

=IF(condition, value_if_true, value_if_false)
Example: =IF(A1>=50,"Pass","Fail") — 60 → Pass
Logical

IFS — Multiple IF

Kai shart ek saath check karta hai.

=IFS(A1>=80,"A",A1>=60,"B",A1>=40,"C",TRUE,"F")
Example: Marks 75 → Grade B
Logical

AND — Aur

Sab shart puri hon tab TRUE.

=AND(A1>50, B1="Pass")
Example: Marks>50 AND Attendance>75% → TRUE/FALSE
Logical

OR — Ya

Koi bhi ek shart puri ho tab TRUE.

=OR(A1="Lahore", A1="Karachi")
Example: City Lahore ya Karachi ho → TRUE
Logical

NOT — Nahi

TRUE ko FALSE aur FALSE ko TRUE karta hai.

=NOT(A1="Absent")
Example: "Present" → TRUE | "Absent" → FALSE
Logical

IFERROR — Galti Chupaao

Error aaye to custom message deta hai.

=IFERROR(VLOOKUP(...),"Not Found")
Use: #N/A ya #DIV/0! ki jagah apna message likhne ke liye.
Logical

ISBLANK — Khali Hai?

Cell khali hai ya nahi check karta hai.

=ISBLANK(A1)
Example: Khali cell → TRUE | Data ho → FALSE
Logical

ISNUMBER / ISTEXT

Cell mein number ya text hai check karta hai.

=ISNUMBER(A1)
=ISTEXT(A1)
Use: Data validation ke liye bahut kaam aata hai.

🔍 Lookup & Reference Formulas

Lookup

VLOOKUP — Upar Se Dhundho

Column mein se koi value dhundh ke uske sath wali value deta hai.

=VLOOKUP(lookup_value, table_array, col_index, [range_lookup])
Example: =VLOOKUP("A101",A:D,3,0) — Roll No se Name dhundho
Lookup

HLOOKUP — Barabar Dhundho

Row mein se value dhundh ta hai.

=HLOOKUP(lookup_value, table_array, row_index, [range_lookup])
Use: Jab data rows mein ho columns mein nahi.
Lookup

XLOOKUP — Nayi Dhundh (Modern)

VLOOKUP ka behtareen replacement.

=XLOOKUP(lookup_value, lookup_array, return_array)
Example: =XLOOKUP("Tahir",A:A,B:B) — Name se ID lo
Lookup

INDEX + MATCH — Powerful Combo

VLOOKUP se zyada flexible lookup.

=INDEX(return_range, MATCH(lookup_value, lookup_range, 0))
Example: Kisi bhi column se kisi bhi taraf dhundho
Lookup

MATCH — Position Dhundho

Value ki position (row/column number) deta hai.

=MATCH(lookup_value, lookup_array, [match_type])
Example: =MATCH("Tahir",A1:A10,0) = 3 (3rd row mein hai)
Lookup

OFFSET — Shift Karo

Starting cell se kisi number of rows/cols door ki cell deta hai.

=OFFSET(reference, rows, cols, [height], [width])
Example: =OFFSET(A1,2,1) = C3 ki value
Lookup

CHOOSE — Intikhaab

Index number ke mutabiq value return karta hai.

=CHOOSE(index_num, val1, val2, val3, ...)
Example: =CHOOSE(2,"Jan","Feb","Mar") = Feb
Lookup

ROW / COLUMN

Cell ki row ya column number deta hai.

=ROW(A5) → 5
=COLUMN(C1) → 3
Use: Dynamic numbering ya conditional formatting mein.

📅 Date & Time Formulas

Date

TODAY — Aaj ki Tarikh

Aaj ki date automatically deta hai.

=TODAY()
Example: =TODAY() = 24/06/2026 (automatically update hoti hai)
Date

NOW — Abhi ka Waqt

Current date aur time deta hai.

=NOW()
Example: =NOW() = 24/06/2026 10:30
Date

DATE — Tarikh Banao

Year, Month, Day se date banata hai.

=DATE(year, month, day)
Example: =DATE(2026,6,24) = 24-Jun-2026
Date

DAY / MONTH / YEAR

Date se din, maah ya saal nikalta hai.

=DAY(A1) | =MONTH(A1) | =YEAR(A1)
Example: Date 24/06/2026 → DAY=24, MONTH=6, YEAR=2026
Date

DATEDIF — Farq Nikalo

Do dates ke beech ka farq nikalti hai.

=DATEDIF(start_date, end_date, "Y")
Example: Umar nikalne ke liye: =DATEDIF(DOB, TODAY(), "Y")
Date

WEEKDAY — Din kaunsa

Date ka weekday number deta hai (1=Sun, 7=Sat).

=WEEKDAY(date, [return_type])
Example: =WEEKDAY(TODAY()) = 4 (Wednesday)
Date

NETWORKDAYS — Kaam ke Din

Weekend aur chutti chodte hue working days ginne ke liye.

=NETWORKDAYS(start_date, end_date, [holidays])
Use: Project deadline calculate karne ke liye.
Date

EDATE — Maah Baad

Kisi tarikh se n maah baad/pehle ki date deta hai.

=EDATE(start_date, months)
Example: =EDATE(TODAY(),3) = 3 mahine baad ki date

💰 Financial Formulas

Finance

PMT — Qist Nikalo

Loan ki monthly installment nikalti hai.

=PMT(rate, nper, pv)
Example: =PMT(0.01,12,-100000) = Monthly installment of 1 Lac loan
Finance

FV — Mustaqbil ki Qeemat

Investment ki future value nikalti hai.

=FV(rate, nper, pmt, [pv])
Use: 5 saal mein savings kitni ho gi?
Finance

PV — Maujuda Qeemat

Future payments ki present value.

=PV(rate, nper, pmt)
Use: Investment return analysis ke liye.
Finance

NPV — Munafa Andaza

Net present value of cash flows.

=NPV(rate, value1, value2, ...)
Use: Business investment decision ke liye.
